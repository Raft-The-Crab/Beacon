# Beacon+ Chat Bubble Image Prompts

Use these prompts to generate overlay textures/backgrounds for Beacon+ bubble skins.
Target output: transparent PNG, 1024x1024, no text, no logos, seamless tile-friendly composition.

## 1. Aurora Pulse
Prompt:
"Abstract aurora light ribbons flowing diagonally, cool cyan and violet glow, soft volumetric haze, premium sci-fi UI texture, subtle depth, high contrast highlights, transparent background, no characters, no text, no logo, clean edges, game UI quality"

Style notes:
- Palette: #38bdf8, #8b5cf6, #22d3ee
- Keep center area less busy so message text stays readable.

Prompt:
"Dark carbon fiber micro pattern with neon cyan rim light accents, premium cyber UI panel texture, subtle diagonal weave, low-key reflections, transparent background, no text, no logo, no symbols, cinematic but clean"

## Export Checklist
- Format: PNG with alpha channel
- Resolution: 1024x1024
- Compression: lossless or near-lossless
- Safety: no copyrighted logos, trademarks, or characters
